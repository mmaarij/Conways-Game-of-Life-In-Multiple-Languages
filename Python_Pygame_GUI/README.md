# Conway's Game of Life Implemented as a Python Graphical Application using Pygame

## Getting Started

### Install Pygame

```console
$ pip install pygame
```

### Run the program

```console
$ python3 main.py
```

## Usage

| Key               | Action                                   |
|-------------------|------------------------------------------|
| Right Mouse Click | Toggle cell state between alive and dead |
| Space             | Start/Stop the simulation                |
| C                 | Clear the board                          |
| R                 | Randomize the board                      |
